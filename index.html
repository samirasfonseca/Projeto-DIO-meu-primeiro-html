Oque são Clients?

São dispositivos que se conectam na internet.

Quando navegamos na internet os servidores nos retornam arquivos estáticos

Aplicações WEB são soluções criadas que possuem a internet como meio de comunicação
cliente x server não sendo necessária a sua instalação

Oque são Servers?

São computadores ou sistemas de computação que fornecem recursos,
dados, serviços ou programas a outros computadores,
conhecidos como clientes, através de uma rede

Oque é HTML?

É uma linguagem de marcação essencial para estruturar conteúdos nas paginas Web.

Curiosidades do HTML!

Define a estrutura:

Define como o conteúdo deve ser organizado,


como parágrafos, títulos, listas, imagens, etc.

Não é uma linguagem de programação:

Não permite criar funcionalidades dinâmicas ou comportamentos interativos.


Para isso, são utilizadas outras linguagens como JavaScript.

Utiliza tags:

As tags são elementos que envolvem o conteúdo e fornecem informações
sobre o seu tipo e estrutura.

É a base para a web:

Sem o HTML, os navegadores não teriam uma estrutura


a partir da qual exibir o conteúdo de uma página.

As principais tags HTML são aquelas que constroem a estrutura básica de uma página web.

Algumas das TAGS mais importantes são:
, head, body, title, h1 a
h6, p, a, img, div, span, ul, ol, li, table, tr, td e th.

Em resumo, o HTML é a linguagem que define a estrutura e o conteúdo de uma página web,
permitindo que ela seja exibida no navegador de forma organizada e compreensível.
